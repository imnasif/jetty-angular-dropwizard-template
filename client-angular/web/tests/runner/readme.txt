This folder contains associated node packages for javascript test engine Jasmine & Test runner Karma.

You need to download and install nodejs first: https://nodejs.org/en/download/

After downloading node js, open a command prompt window in this location and run:

npm install

This will install necessary node packages for jasmine & karma.

Additionally, install the following packages globally: 


karma-cli - makes karma available in command line

npm install -g karma-cli
